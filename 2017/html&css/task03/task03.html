昨日，徐小平在其真格基金的 CEO 交流群里分享了对区块链技术的看法，随即聊天截图被挂得满天飞，媒体报道也接踵而来。

徐小平说，不要临渊羡鱼，而是要冲到浪潮中去，迎接区块链带给各行业的冲击。

借用徐小平的这句话看，游戏行业确实已经冲到了区块链的浪潮中去了。

近几周以来，国内不少游戏公司或多或少地都在对区块链技术进行试探，有公司效仿海外游戏《CryptoKitties》推出了区块链游戏，有公司谨慎地测试了区块链游戏频道。

但不管试探的行为如何，试探的短期反馈是，这些公司的股价都大涨，涨停者更是不在少数。

尽管数字虚拟货币还未得到国内监管层的明确认可，但不可否认的是，“区块链 + 游戏” 已经成为了 2018 年游戏行业第一个风口。
